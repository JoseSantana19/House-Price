# House-Price
Exercício Resolvido de Machine Learning: Regressão Linear Múltipla em Python

## Objetivo do Exercício

Este exercício tem como finalidade realizar uma análise exploratória da base de dados de precificação de imóveis e seus atributos, identificar relações entre as variáveis e, em seguida, treinar um modelo de regressão linear múltipla para prever o preço das casas com base em suas características.
